❓ ¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcas de Hipertexto). No es un lenguaje de programación, sino un lenguaje de maquetación que funciona como el esqueleto o la estructura de cualquier página web.

Imagina que una web es un edificio: HTML define dónde van las columnas, las paredes, las ventanas y las puertas. Se escribe utilizando etiquetas de apertura y cierre (como <p> y </p>) para decirle al navegador qué parte del texto es un título, qué es un párrafo o dónde debe colocarse una imagen.

🏗️ Explicación de Etiquetas HTML

A continuación se detallan las etiquetas fundamentales junto con su respectiva sintaxis de cierre, esenciales para que el navegador interprete correctamente el documento:

<html> ... </html>   : Etiqueta raíz que envuelve y encierra todo el contenido del documento web.
<head> ... </head>   : Contenedor técnico que encierra los metadatos, el título y los estilos de la página.
<body> ... </body>   : Bloque principal que encierra todos los textos, imágenes y elementos visibles de la web.
<h1> ... </h1>       : Encabezado de nivel 1. Define y cierra el título principal de mayor importancia.
<p> ... </p>         : Delimita y cierra un bloque de texto correspondiente a un párrafo independiente.
<img src="...">       : Inserta un recurso gráfico (es una etiqueta especial autoconclusiva, no requiere </img>).
<a href="..."> ... </a> : Abre, contiene el texto interactivo y cierra un enlace o hipervínculo de navegación.
<ul> ... </ul>       : Define y cierra la estructura completa de una lista de elementos no ordenada (viñetas).
<li> ... </li>       : Delimita y cierra cada uno de los elementos o ítems individuales dentro de una lista.

🎨 ¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) es el lenguaje informático de diseño que se encarga de interceptar el esqueleto plano de etiquetas generado por HTML para dotarlo de una capa estética profesional.

A través de CSS, los desarrolladores controlamos de forma unificada la apariencia global de un sitio: la distribución del espacio, los colores de contraste, las cajas translúcidas y los efectos interactivos que enriquecen la navegación de los usuarios.
