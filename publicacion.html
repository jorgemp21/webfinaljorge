🚀 Publicación en Servidores con GitHub Pages

Para la puesta en producción y publicación en Internet de esta web de forma libre, se ha empleado la infraestructura en la nube de GitHub:

  • GitHub: Actúa como nuestro software de control de versiones y almacenamiento de código en la nube a través de repositorios remotos seguros.
  • GitHub Pages: Herramienta de alojamiento estático automatizado que transforma de forma inmediata los archivos del repositorio en un sitio web público a través de un enlace.
